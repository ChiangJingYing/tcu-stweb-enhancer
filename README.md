# ✨ TCU StWeb 網頁優化擴充套件 (TCU StWeb Tools)

這是一個專為 **慈濟大學 StWeb 系統** (`admin.tcu.edu.tw`) 量身打造的 Chrome 擴充套件，旨在優化使用者介面 (UI) 並自動化繁瑣的查詢流程，讓您的工作更是得心應手！ 🚀

---

## 🎯 核心功能 (Core Features)

### 1. 🖥️ 介面自動優化 (UI Enhancement)
當您進入系統主頁面 (`Stmain.php`) 時，套件會自動執行以下優化：
*   📅 **自動日程查詢**：自動將查詢範疇設定為**當前月份**，並自動送出請求抓取付款資訊，直接替換原有的無用區塊（如反詐、禁菸宣導），讓您一進首頁就能看到當月學校付款資訊！
*   🔗 **整合 iCan 入口**：在介面中貼心嵌入 **「開啟 iCan」** 快捷按鈕，一鍵切換系統，不再需要迷航於各個分頁中。
*   🧭 **修正導覽連結**：修復左上角 Logo 連結，點擊即可正確重新整理並回到 `Stmain.php` 主頁面，無需再從複雜的下拉選單中尋找入口。

### 2. 📌 智慧釘選系統 (Smart Pinning System)
打造屬於您個人的快捷工作區！

*   **🔘 浮動功能按鈕 (Floating Button)**
    *   右下角提供一個顯眼的藍色圖釘按鈕 (📌)。
    *   🖐️ **自由拖曳**：按鈕擋到視線了嗎？您可以將它拖曳至畫面任何角落，位置會**自動記憶**，下次登入依然在您最順手的位置。

*   **✨ 新增釘選 (Selection Mode)**
    1.  點擊圖釘按鈕展開選單。
    2.  點擊 **「➕ 新增功能 (Add Pin)」** 進入選擇模式（游標將變為十字 ➕）。
    3.  直接點擊畫面上任何您常用的**功能按鈕** (例如「請購單申請」、「成績查詢」等)。
    4.  ✅ 該功能會立即被「釘選」至下方的快捷區域，方便下次秒速訪問！

*   **📋 管理釘選清單**
    *   在浮動選單中，您可以概覽所有已釘選的項目。
    *   點擊項目旁的 **「✕」** 按鈕即可移除不需要的釘選。

*   **✏️ 快捷區域編輯 (Edit Mode)**
    *   點擊快捷區域右上角的 **「✎」編輯按鈕** 即可進入編輯模式。
    *   🔃 **拖曳排序**：長按並拖曳即可調整順序，藍色指標線會引導您放置的精確位置。
    *   🗑️ **快速刪除**：每個項目右上角會出現紅色 **「✕」**，點擊立刻移除。
    *   💾 **完成編輯**：調整完畢後，點擊 **「✓」** 即可儲存並退出。

## 🛠️ 使用範圍與限制

本擴充套件**僅在以下特定情境**生效：

*   📍 **目標網址**：`https://admin.tcu.edu.tw/TCUstweb/Stmain.php`
*   ⚙️ **觸發條件**：
    *   **介面修改**：僅在**乾淨的網址**（無 `?page=...` 等參數）下執行，避免干擾查詢結果頁。
    *   **浮動按鈕**：在所有 `Stmain.php` 頁面（含子分頁）皆會顯示，讓您隨時隨地都能管理釘選。

## 🚀 安裝方式 (Installation)

1.  📦 **下載檔案**：確認您已擁有本專案資料夾 (`TCU-stWeb_tools`)。
2.  🧩 **開啟擴充功能管理**：
    *   在 Chrome 網址列輸入 `chrome://extensions/` 並按下 **Enter**。
3.  ⚙️ **啟用開發者模式**：
    *   開啟頁面右上角的 **「開發人員模式 (Developer mode)」** 開關。
4.  📂 **載入套件**：
    *   點擊左上角的 **「載入未封裝項目 (Load unpacked)」** 按鈕。
5.  ✅ **完成安裝**：
    *   選擇 `TCU-stWeb_tools` 資料夾。套件即刻啟用，請前往 StWeb 系統體驗！

---

## ⚠️ 注意事項

*   🛡️ **隱私安全**：本套件僅為前端介面優化工具，程式完全運行於您的瀏覽器端，**絕不會**蒐集任何個人帳號、密碼或數據，請安心使用。
*   🔧 **維護提醒**：若學校網頁結構大幅改版 (例如 XPath 變更)，可能會導致自動填入功能暫時失效，屆時需更新擴充套件代碼。
