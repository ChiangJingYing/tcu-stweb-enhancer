# TCU StWeb 網頁優化擴充套件 (TCU StWeb Tools)

這是一個專為慈濟大學 StWeb 系統 (`admin.tcu.edu.tw`) 設計的 Chrome 擴充套件，旨在優化使用者介面並自動化部分繁瑣的查詢流程。

## 🎯 核心功能

### 1. 介面自動優化
當您進入系統主頁面 (`Stmain.php`) 時，套件會自動執行以下優化：
*   **自動填入日期查詢**：自動將查詢範疇設定為當前月份，並自動送出 POST 請求至 `acc/stMscQry.php` 抓取資料，直接替換原有的反詐、禁菸區塊。
*   **整合 iCan 入口**：在介面中嵌入「開啟 iCan」的快捷按鈕，方便快速切換系統，減少點擊跳轉次數。
*   **修正頂部導覽連結**：新增左上角 Logo 連結，使其正確導回 `Stmain.php` 主頁面，方便快速跳轉回主頁面（原需要點擊下拉選單再點擊主頁面）。

### 2. 自定義常用功能釘選 (Pinning System)
*   **浮動功能按鈕 (Floating Button)**：右下角提供一個可拖曳的藍色圖釘按鈕 (📌)。
    *   **拖曳功能**：您可以將按鈕拖曳至畫面任何角落，位置會自動儲存，下次重新整理頁面時會停留在最後放置的位置。
*   **新增釘選 (Selection Mode)**：
    1. 點擊圖釘按鈕展開選單。
    2. 點擊「新增功能 (Add Pin)」進入選擇模式（游標變為十字且會有提示）。
    3. 點擊畫面上任何您想釘選的功能按鈕 (例如「請購單申請」等)。
    4. 該功能會立即被釘選至下方的快捷區域。
*   **管理釘選清單**：
    *   在浮動選單中，您可以查看所有已釘選的項目。
    *   點擊項目旁的「✕」按鈕即可移除該釘選。
*   **快捷區域同步**：您釘選的項目會即時顯示在頁面下方的快捷區域中。
*   **快捷區域編輯 (Edit Mode)**：
    *   在主頁面下方的快捷區域，點擊右上角的「✎」編輯按鈕即可進入編輯模式。
    *   **拖曳排序**：可直接拖曳調整釘選項目的順序，拖曳時會有藍色游標顯示插入位置。
    *   **快速刪除**：每個項目右上角會出現「✕」按鈕，點擊即可刪除。
    *   完成後點擊「✓」按鈕即可退出編輯模式。

## 🛠️ 使用範圍與限制

本擴充套件**僅在以下特定網址**生效：

*   **目標網址**：`https://admin.tcu.edu.tw/TCUstweb/Stmain.php`
*   **觸發條件**：
    *   **介面修改**：僅在**乾淨的網址**（無任何查詢參數，例如 `?page=...`）下才會執行，以避免干擾查詢結果頁面的顯示。
    *   **浮動按鈕**：在所有 `Stmain.php` 頁面（包含子查詢頁面）皆會顯示，方便隨時管理釘選。

## 🚀 安裝方式

1.  **下載/準備檔案**：確認您已擁有本專案資料夾 (`TCU-stWeb_tools`)。
2.  **開啟 Chrome 擴充功能管理頁面**：
    *   在網址列輸入 `chrome://extensions/` 並按下 Enter。
3.  **啟用開發人員模式**：
    *   開啟頁面右上角的 **「開發人員模式 (Developer mode)」** 開關。
4.  **載入未封裝項目**：
    *   點擊左上角的 **「載入未封裝項目 (Load unpacked)」** 按鈕。
5.  **選擇資料夾**：
    *   選擇下載後的`TCU-stWeb_tools`資料夾。
6.  **完成**：
    *   擴充套件現在已啟用。您可以前往登入系統進行測試。

---

## ⚠️ 注意事項

*   本套件僅為前端介面優化工具，不會蒐集任何個人帳號密碼或資料。
*   若網頁結構改版 (XPath 變更)，可能會導致自動填入功能失效，需更新程式碼中的選取器。
